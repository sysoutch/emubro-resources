Edit Game Properties

Use game details to tune launch behavior and metadata quality.

Updated: 2026-02-23

Recommended fields

  • Platform short name and emulator path alignment.
  • Game code/serial for better cover lookup and matching.
  • Image path and install status for cleaner library display.

Launch options

  • Choose default launcher policy for launcher-backed entries.
  • Set run-as mode where required (admin/user) on Windows.

When editing in bulk

  • Apply consistent naming and tags per platform.
  • Avoid changing platform values unless you also verify emulator mapping.
