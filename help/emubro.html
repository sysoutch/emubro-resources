emuBro Help Center

This section explains the main app areas, where to click first, and how to solve common setup issues quickly.

Updated: 2026-02-23

Getting started

  • Use the top navigation or side rail to open Library, Tools, Support, and Community.
  • Set your library paths in Settings first so scanning and imports use the correct folders.
  • If you use portable + setup installs, point both to the same emubro-resources path in Settings.

Core workflow

  • Scan/import games and emulators.
  • Open game details to adjust metadata, launcher settings, and run-as behavior.
  • Use filters/sort/grouping to keep large libraries readable.
  • Use Support (General Chat or Troubleshoot) for guided help.

Need help fast

  • If a game does not launch, check emulator path, platform match, and file extension support.
  • If artwork is missing, use Cover Downloader and confirm game serial/code where needed.
  • For launcher games (Steam/Epic/GOG), verify launcher install path and launch policy.
