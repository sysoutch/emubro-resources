Find and Download Covers

Download missing covers using configured source URLs and game serial/code matching.

Updated: 2026-02-23

Current support

  • PS1 (PSX) and PS2 sources by serial/code.
  • Single-game download from game details and batch download from Tools.

Best results

  • Ensure game code/serial is set correctly.
  • Review source URL templates in config if your preferred source changed.

Failure handling

  • 404 usually means wrong/missing serial or unavailable asset.
  • Try alternate source links if configured.
