Help Us Improve

Feedback and issue reports help stabilize releases and improve platform support.

Updated: 2026-02-23

Useful bug reports include

  • Exact steps to reproduce.
  • Platform/OS/emulator details.
  • Error text and relevant logs/screenshots.

Feature requests

  • Describe the user goal first, then expected behavior.
  • Include edge cases (launcher games, Linux packaging, portable mode).
