Configure Platforms

Platform config controls supported file types, emulator recommendations, and launcher integration metadata.

Updated: 2026-02-23

Key fields

  • Platform short name and display name.
  • Supported image/archive/file extension lists.
  • Recommended emulators and optional installer hints.

Launcher/platform integrations

  • PC platform can include launcher discovery/launch policies.
  • Use consistent keys for Steam/Epic/GOG integration data.
