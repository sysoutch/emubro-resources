About Emulator Entries

An emulator entry in emuBro is a launch profile bound to a platform and executable.

Updated: 2026-02-23

Entry contents

  • Display name, platform short name, executable path, optional start parameters.
  • Install metadata (where available), including Linux installer hints.

Why multiple entries can help

  • Keep one stable profile and one experimental profile for the same platform.
  • Switch quickly when testing compatibility/performance.
