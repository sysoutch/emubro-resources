Add Emulator

Create a new emulator entry and bind it to the correct platform.

Updated: 2026-02-23

Steps

  • Pick executable (or app image/binary depending on OS).
  • Set platform short name exactly as expected by your game entries.
  • Save and test launch with one known-good game.

Validation tips

  • Use absolute paths when possible.
  • Verify the executable launches independently before adding to emuBro.
