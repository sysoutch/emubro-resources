Emulators

Configure emulator executables, launch args, and per-platform compatibility.

Updated: 2026-02-23

Core requirements

  • Each platform should map to at least one valid emulator executable.
  • Executable paths must exist and be accessible by the current user.
  • Platform file type support should match your game files.

Installation sources

  • Local manual installation.
  • Configured package manager paths (including Linux package metadata where available).

Troubleshooting

  • If launch opens but game fails, check command arguments and BIOS requirements.
  • If no emulator is found, re-open emulator config and verify platform mapping.
