Updates

Manage app updates and emubro-resources updates separately.

Updated: 2026-02-23

App updates

  • Check, download, and install via updater controls.
  • Channel/version metadata is shown when available.

Resources updates

  • Resources can update independently from app binaries.
  • Manifest/commit-based checks depend on configured repository strategy.

If update fails

  • Check network/release asset availability.
  • Retry after cleaning broken partial downloads.
