Remove Games

Remove entries safely from the library while controlling file deletion behavior.

Updated: 2026-02-23

Before removing

  • Confirm whether you want to remove only the DB entry or also delete files.
  • For launcher games, removal usually only affects library records.

Safe cleanup flow

  • Filter by platform/tag to verify selection.
  • Remove in smaller batches when cleaning large duplicate sets.
  • Rescan if you want to re-import corrected entries later.
