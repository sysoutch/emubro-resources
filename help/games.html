Games Library

Manage game entries, metadata, platform mapping, launch behavior, and organization.

Updated: 2026-02-23

What you can edit

  • Name, description, genre, rating, install state, tags, and image.
  • Platform and emulator mapping for launch.
  • Run options such as run-as admin/user on supported systems.

Best practices

  • Keep platform short names consistent (for filters and tooling).
  • Prefer clean file names and set game codes/serials where available.
  • Use grouping for duplicates and regional variants.

Troubleshooting

  • If launch fails, recheck emulator executable path and game path validity.
  • If game was moved, use relink/search missing file actions.
